功能

自建 pillui 组件库，纯 Pillow Canvas 渲染，轻量高效

🔑

SSH 密钥连接

基于 ed25519/RSA 密钥的安全连接，支持密码回退，自动管理 known_hosts。

🔄

IP 漂移自动修复

树莓派 DHCP 更换 IP 时自动重新解析主机名并重连，指数退避最多 3 次。

📊

实时状态监控

CPU、温度、内存、磁盘、运行时间、负载，侧边栏 + 详情双视图。

📁

双栏文件管理器

远程/本地双栏，上传、下载（含进度条）、删除、重命名、新建文件夹。

⚡

远程执行

右键运行 Python/Shell 脚本，结果实时返回或推送到终端。

💻

流式命令终端

多标签独立会话，流式/阻塞双模式，Ctrl+C 中断，命令历史。

🎨

主题 & 背景

深色/浅色模式，绿/蓝/深蓝三色强调。任意图片背景，四种适配模式。

⚡

快捷命令

10 个预设命令一键发送到终端，提升操作效率。

🪶

极致轻量

原生桌面应用，pillui 自建组件库，Canvas 直接渲染，内存占用极低。

🔧

开箱即用

仅依赖 Python 3.9+、paramiko、Pillow，Windows 一键安装包，无需手动配置。

三步安装

1

下载安装包

点击上方下载按钮，获取 PiManager-2.2.2-setup.exe（32 MB）。

2

运行安装程序

双击安装包，按提示完成安装。自动创建桌面快捷方式。

3

连接树莓派

输入主机名或 IP，选择 SSH 密钥（自动发现 ~/.ssh 目录），一键连接。

ssh-keygen -t ed25519 -C "pimanager"

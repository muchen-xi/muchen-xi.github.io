## 已有作品

### ♟ 五子棋 AI
完整棋盘逻辑 + 简单的评估函数 AI。虽然比不上 AlphaGo，但亲眼看到自己写的代码"会下棋"的那一刻，非常奇妙。

### 🎲 PyGame 小游戏
几个练手项目：打砖块、贪吃蛇、简易弹幕射击。每一行代码都在理解游戏循环、碰撞检测、状态管理。

### 🧠 智能下棋机器人
结合硬件——摄像头识别棋盘 → Python 计算落子 → 机械臂执行。软硬件结合的第一次尝试。

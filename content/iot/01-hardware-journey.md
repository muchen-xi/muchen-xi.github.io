# 硬件之路：从 ESP32 到树莓派

> 面包板上的第一颗 LED 亮起来，就停不下来了

---

## 起点：ESP32 校园天气助手

**硬件**：ESP32 + SSD1306 OLED + MAX30102 血氧传感器

**软件**：自研中文点阵字库——因为 OLED 库不支持中文，自己手写了一个 16×16 的点阵映射表。三页 UI 通过物理按键切换（天气 / 血氧 / 关于）。

**收获**：第一次理解 I2C 协议、显存、字库编码。

**成绩**：区二等奖。

## 进阶：树莓派智能驾驶小车

**硬件**：Raspberry Pi Pico + L239D 电机驱动 + 双 18650 电池

**软件**：MicroPython 写 PWM 调速 + 舵机转向 + 超声波避障。

**难点**：PWM 频率和占空比的调试——太高电机啸叫、太低扭矩不够。最后用 50Hz 舵机 + 1000Hz 电机的组合。

**成绩**：区二等奖。

## 现在：智能天气站

**硬件**：Pi Zero W + ILI9341 TFT + GPS + 4G + I2S 音频

这是目前最复杂的项目——SPI 屏、UART GPS、I2S 音频、4G 联网，四路协议同时跑。还有锂电供电、电源管理。

---

> 硬件调试的哲学：**拆成最小单元，一个一个验证**。

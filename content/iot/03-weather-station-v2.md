## v2.0 目标

### 显示升级
ILI9341 TFT + Pillow 绘制天气 UI：
- 当前天气大图标 + 温度大字
- 24h 温度趋势折线图
- 空气质量环形进度条
- 观星指数星级评价

### 语音交互
- DeepSeek API → "今天要带伞吗？" → 语音回答
- 定时语音播报（早上 7:00 "今日天气..."）
- I2S 麦克风录音 → 语音识别 → AI 对话

### 联网升级
- 4G 上网卡托 → 脱离 WiFi 独立运行
- 实时上传数据到云服务器
- 手机端 Web 面板查看

### 电源升级
- ETA9742 电源管理模块
- 18650 × 2 并联（~6000mAh）
- 5V 过压保护
- 目标续航：12h+

---

> v2.0 的灵魂不是更多传感器，是 **能用语音聊天气**。
